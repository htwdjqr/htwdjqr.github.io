Spring Cloud（6）——注册中心高可用集群

发表于 2017-11-30 | 分类于 Spring Cloud |

在Spring Cloud（1）——服务注册中心这篇文章中，我们已经搭建好一个单机的注册中心。这篇文章要做的就是把单机版的注册中心改造为高可用集群模式。

我们可以创建三个注册中心节点，每个节点进行两两注册，实现完全对等的效果，可以达到集群的最高可用性，任何一个节点挂掉都不会影响服务的注册与发现。
